# QuestRaid: The Global Decentralized Mastery Marketplace

## ⚡ Vision
QuestRaid is a high-trust, RPG-style gig economy platform. It connects "Quest Givers" (Customers) with "Runners" (Players) for local service fulfillment (Delivery, Errands, Skilled Trades).

## 🎨 Design System: "Obsidian Surge"
- **Primary Background:** #020617 (Obsidian Deep Slate)
- **Primary Accent:** #003e78 (Lightning Blue)
- **Secondary Accent:** #22d3ee (Cyan Glow)
- **Aesthetic:** Premium Stealth / High-Contrast Digital Terminal.
- **Typography:** Geist Sans / Mono (Minimum 16px base font for accessibility).

## 🏗️ Core Architecture
- **Tech Stack:** React (Vite), Tailwind CSS v4, Framer Motion, Firebase (Auth/Firestore).
- **RPG System:** 1 PHP earned = 1 XP. Uncapped leveling with milestone rewards (Lv 5, 10, 50, 100).
- **Mastery System:** 
    - *True-Rated:* Admin-verified (Electrician, Plumber, IT).
    - *Customer-Rated:* Community-vouched (Shopper, Courier).
- **Reputation Engine:** Symmetric ratings for Runners (Public) and Asymmetric/Hidden 'Shadow Ratings' for Givers (Internal only).
- **Logistics:** Tri-Node Distance UI (Runner ➔ Store ➔ Home) and Strict vs. Flexible fulfillment toggles.

## 🗺️ Current Roadmap
- **Sprint 1:** Foundation, Landing Page, and Global UI Shell.
- **Sprint 2:** Firebase Auth, Role Switching (Giver/Runner), and Profile Management.
- **Sprint 3:** Real-time Quest Board, Creation Form, and Core Logic.
- **Sprint 4:** RPG Stats, Leveling, and Wallet UI.
